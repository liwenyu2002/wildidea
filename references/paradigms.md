## 7.1 城市规划与建筑

| 范式 | 一句话本质 |
|------|-----------|
| 混合用途 vs 功能分区 | 空间高度隔离带来效率但杀死活力；混合用途带来混乱但有韧性 |
| 窄马路密路网 vs 宽马路疏路网 | 毛细渗透 vs 主动脉爆发，两种流量组织哲学 |
| 自发生长 vs 总体规划 | 有机城市 vs 光辉城市；核心不是谁更好，而是阶段与控制权不同 |
| 场所精神 Genius Loci | 好空间不是被设计出来的，而是被认出的 |
| 围墙社区 vs 开放街区 | 安全感和流动性不可兼得时的取舍 |
| 临时城市 / 快闪空间 | 不稳定使用不是低级状态，而是探索真实需求的探针 |
| 基础设施先行 | 真正决定城市形态的不是建筑，而是看不见的管线、道路和接口 |

## 7.2 生态学与进化

| 范式 | 一句话本质 |
|------|-----------|
| 先锋物种 → 顶级群落 | 系统演替有阶段，跳过阶段不等于加速 |
| 干扰驱动多样性 | 适度火灾/风暴增加多样性；完全保护反而让系统脆弱 |
| 生态位分化 | 共存的秘密是不完全竞争同一个资源维度 |
| r 策略 vs K 策略 | 大量低投资后代 vs 少量高投资后代 |
| 红皇后假说 | 你必须不断奔跑才能留在原地 |
| 边缘效应 | 两个生态系统交界处常有最高多样性和最强不稳定性 |
| 入侵物种 | 外来系统不一定强，但可能破坏原有约束关系 |
| 共生体 | 个体能力不如关系结构重要 |

## 7.3 博弈、决策与军事

| 范式 | 一句话本质 |
|------|-----------|
| 厚势 vs 实利 | 放弃确定收益换取未来可能性 |
| 囚徒困境 → 重复博弈 | 一次相遇自利最优，长期共存合作最优 |
| 探索 vs 利用 | 在试新路和走老路之间动态分配资源 |
| 最小后悔原则 | 不是选最好结果，而是选最不后悔的 |
| OODA Loop | 观察、判断、决策、行动的循环速度决定竞争力 |
| 诱饵与佯攻 | 不是每个动作都为直接收益，有些是为了改变对方模型 |
| 非对称战争 | 弱者不在强者擅长的维度竞争，而是改变规则 |

## 7.4 艺术、工艺与审美

| 范式 | 一句话本质 |
|------|-----------|
| 约束释放创造力 | 框框是自由的支架，不是牢笼 |
| 留白 / 负空间 | 不画出来的部分比画出来的更重要 |
| 孤品逻辑 | 价值来自不可复制的偏差而非完美复现 |
| 平仄 / 格律 | 形式限制不是限制内容，而是让好内容浮现 |
| 毛坯房越精细越难改 | 前期投入越深，后期转向成本越高 |
| 修复的可见性 | 修复痕迹本身成为物品的新的信息层 |
| 即兴演奏 | 在共享骨架上实时偏离，不是无规则 |

## 7.5 工程、系统与安全

| 范式 | 一句话本质 |
|------|-----------|
| 冗余设计 vs 精益设计 | 多出来的部分是故障时的生存空间 |
| 技术债 | 偷过的懒会利滚利 |
| 完美是足够好的敌人 | 最后 1% 的完美可能消耗 50% 的资源 |
| 安全边际 | 承重必须高于常规需求，因为存在未知未知 |
| 费米估算 | 不需要精确数据也能用数量级判断方向 |
| 默认拒绝 vs 默认允许 | 保护优先还是便利优先的系统哲学 |
| 故障隔离舱 | 不是阻止失败，是限制失败传播 |
| 可观测性 | 不能观测的系统无法可靠改进 |

## 7.6 科学哲学与认知

| 范式 | 一句话本质 |
|------|-----------|
| 奥卡姆剃刀 vs 希卡姆警车 | 简单有优势，但复杂世界多因共存 |
| 相关≠因果 | 常见且重要的认知边界 |
| 证伪 | 科学理论因为能被推翻才成立 |
| 库恩范式转换 | 科学不是线性积累，是旧框架重构 |
| 观察者效应 | 观察方式决定观察结果 |
| 哥德尔不完备 | 自治系统内部有无法自证的真命题 |
| 拉卡托斯研究纲领 | 理论有硬核和保护带，失败不一定立即推翻核心 |
| 多重可实现性 | 同一功能可由多种底层结构实现 |

## 7.1 城市规划与建筑

| 范式 | 一句话本质 |
|------|-----------|
| 混合用途 vs 功能分区 | 空间高度隔离带来效率但杀死活力；混合用途带来混乱但有韧性 |
| 窄马路密路网 vs 宽马路疏路网 | 毛细渗透 vs 主动脉爆发，两种流量组织哲学 |
| 自发生长 vs 总体规划 | 有机城市 vs 光辉城市；核心不是谁更好，而是阶段与控制权不同 |
| 场所精神 Genius Loci | 好空间不是被设计出来的，而是被认出的 |
| 围墙社区 vs 开放街区 | 安全感和流动性不可兼得时的取舍 |
| 临时城市 / 快闪空间 | 不稳定使用不是低级状态，而是探索真实需求的探针 |
| 基础设施先行 | 真正决定城市形态的不是建筑，而是看不见的管线、道路和接口 |

## 7.2 生态学与进化

| 范式 | 一句话本质 |
|------|-----------|
| 先锋物种 → 顶级群落 | 系统演替有阶段，跳过阶段不等于加速 |

| 入侵物种 | 外来系统不一定强，但可能破坏原有约束关系 |
| 共生体 | 个体能力不如关系结构重要 |

## 7.3 博弈、决策与军事

| 范式 | 一句话本质 |
|------|-----------|
| 厚势 vs 实利 | 放弃确定收益换取未来可能性 |
| 囚徒困境 → 重复博弈 | 一次相遇自利最优，长期共存合作最优 |
| 探索 vs 利用 | 在试新路和走老路之间动态分配资源 |
| 最小后悔原则 | 不是选最好结果，而是选最不后悔的 |

| 诱饵与佯攻 | 不是每个动作都为直接收益，有些是为了改变对方模型 |
| 非对称战争 | 弱者不在强者擅长的维度竞争，而是改变规则 |

## 7.9 深海热泉与极端微生物

| 范式 | 一句话本质 |
|------|-----------|
| 化能自养 | 没有阳光也能靠化学能支撑整个生态系统——基础能量来源往往不是你最熟悉的那个 |
| 热液喷口间歇性 | 生产力不匀速，是爆发—休眠的周期律——资源丰富时疯长，没有时蛰伏 |
| 嗜极生存策略 | 极端环境不是错误，是特定生命形式的天然栖息地——条件对另一个物种来说是"正好" |
| 共生互养 | 深海的物种依赖不是"大+小"，是化学物质交换——谁的废物是另一方的养料 |
| 缓慢生长但长寿 | 深海生物代谢极慢但极长寿——速度和质量经常不可兼得 |
| 生物发光 | 在完全黑暗中，最有效的沟通不是"大喊"而是"点亮"——信号形式必须匹配环境感知通道 |

## 7.10 中世纪修道院与手稿生产

| 范式 | 一句话本质 |
|------|-----------|
| 抄写即修习 | 重复不是低效——手抄一遍本身就是内化过程，速读/复印反而失去理解 |

| 羊皮纸的稀缺性 | 一张羊皮只能产几页纸——资源约束决定内容是"该写什么"而非"能写什么" |
| 伪经与正典 | 早期教会花上百年争论哪些文本应该保留——筛选机制比创作机制更决定最终形态 |

## 7.12 布须曼人狩猎采集

| 范式 | 一句话本质 |
|------|-----------|
| "共享肉"非"分配肉" | 猎杀大型动物后，猎人不拥有肉——肉归群体，由最年长的女性决定怎么分。谁猎到的不重要 |
| 追踪即推理 | 从一个扭曲的草叶形状推断三个小时前羚羊的经过路线——不是"看脚印"是"读故事" |
| 平等主义压制 | 如果有人开始炫耀，群体用戏弄/忽略施压让他收敛——主动防治等级化 |
| 3天工作制 | 布须曼人每周采集/狩猎约15小时吃饱，剩余时间社交讲笑话——"效率"定义本身是文化的 |
| 跟随水源移动 | 不固定居所——好策略是"知道什么时候离开"而不是"加固当前据点" |
| 干旱知识储备 | 不囤积食物（会坏），囤积"哪里能找到水源的信息"——信息储备比实物储备更持久 |

## 7.13 赌场与概率系统

| 范式 | 一句话本质 |
|------|-----------|
| 房内优势 House Edge | 每个赌局设计时嵌入一种微小的数学偏差——长期看这个偏差就是全部利润 |
| 赌徒谬误 | 人类天生相信"该轮到我赢了"——用过去结果预测独立事件，是最顽固的认知偏差 |
| 筹码隔离 | 用筹码代替真实货币——增加心理距离让人更容易下注 |
| 抽水 vs 对赌 | 稳定金流来自抽取每一手（庄家坐收），不是靠押注结果——不做赌博的一方永远不输 |
| 止损（Stop-Loss）边界 | 好赌场给玩家设输钱上限——保护能让你再回来 |
| 老虎机的奖励调度 | 最佳成瘾结构不是"赢了就给"，是"接近赢"（near miss）比真正赢了更让人想继续 |
| 点数/积分锁定 | 酒店积分/免费房间不是礼物——是让用户不那么容易离开的锚 |

## 7.14 古典音乐与声学

| 范式 | 一句话本质 |
|------|-----------|
| 对位法 | 两条独立旋律同时进行且各自保持独立，但合在一起好听——不是"统一"是"不冲撞的独立" |
| 调性建立与打破 | 先确立一个稳定的"家"（主调），偏离产生张力，回归释放——听众想知道的是"怎么回来"不是"去了哪" |
| 休止符 | 音乐中最重要的部分是无声的部分——节奏由无声定义 |
| 泛音列 | 每个音符携带一串听不见的倍频——最大影响来自超出感知范围的结构 |
| 指法经济（Fingering） | 好指法不追求"最优手指"，追求"最省心转移到下一个音符"——局部效率必须服从全局衔接 |
| 排练与上座率 | 最高质量的演出，99%的时间和听众无关——产出端的可见部分远小于幕后的不可见部分 |
| 音乐会礼仪 | 限制咳嗽/鼓掌时机——自由在固定结构中才有意义 |

## 7.15 免疫系统

| 范式 | 一句话本质 |
|------|-----------|
| 免疫记忆 | 最好的防御不是第一次就打赢，是记住对方下次更快识别——记忆比暴力更高效 |
| 自身免疫错误 | 系统攻击自己的时候不是"发疯了"，是识别逻辑出了微小偏差——精准识别比精确打击难 |

## 7.16 中世纪炼金术与占星

| 范式 | 一句话本质 |
|------|-----------|
| 对应论（Correspondences） | 万物之间存在隐秘关联——星球→金属→器官→情绪（火星→铁→肌肉→愤怒）——不是胡说，是跨域映射的朴素版本 |
| 哲人石    | 认为存在一种物质能完成一切转化——对"万能解决方案"的执念本身有巨大机会成本 |
| 密封加热 — Solve et Coagula | "溶解与凝固"：先打破原有结构再重组——不拆解就无法纯粹化 |
| 蒸馏纯化    | 反复蒸腾→冷凝→再蒸腾——纯度来自循环，不是单次过滤 |
| 点石成金的意象 | 把低价值物质转化为高价值——不是"增加"，是"改变材料性质" |
| 占星时机的选择 | 不是事情做不做的问题，是什么时候做的问题——时机本身是一种工艺参数 |
| 炼金术的保密行话 | "绿狮吞噬太阳" 意思是金被王水溶解——保密不是为了神秘，是为了控制知识传播速度 |

## 7.17 极地探险与远征物流

| 范式 | 一句话本质 |
|------|-----------|
| Shackleton 的选择 | 船沉了不救船救船员——放弃资产优先保人，长期收益更高 |
| 补给站（Depot）前置 | 出发前先在沿途埋好回程食物，而不是指望回程时还有——真正的储备不是在起点而是在中途 |

| 南极点竞赛 Amundsen vs Scott | Amundsen用狗（已有经验）、Scott用马（新技术）——不要在新地方同时试用新和新方法 |
| Crevasse 裂缝规避 | 冰盖表面看似平坦但下面有深不见底的裂缝——最好走的路线下面可能就是最危险的 |

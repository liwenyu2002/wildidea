# WildIdea HTML 海报配色方案

默认只使用这一版暖白风格。除非用户明确要求，不要回到深色、霓虹、标题渐变或多色卡片主题。

## 默认色

| 用途 | 颜色 |
|------|------|
| 浏览器背景 | `#f3eee4` |
| 画布背景 | `#fbf7ef` |
| 卡片背景 | `#fffaf2` |
| 主文字 | `#211f1c` |
| 弱文字 | `#766b60` |
| 棕色强调 | `#8a4f32` |
| 标题棕 | `#7c4a2d` |
| 标签背景 | `#ead1ba` |
| 虚线隔离区 | `#d9c7b6` |
| 卡片边框 | `#dfd1c2` |
| 卡片顶线 | `#b46b45` 到 `#d7a86e` |

## 使用规则

- 画布保持白底米黄，不做大面积渐变背景。
- 标题不用渐变文字，直接用深棕黑。
- 卡片全部使用同一暖白底色，靠来源标签和文字层级区分内容。
- `.desc` 是卡片里最大的正文区，用于承载“用户领域怎么干（通俗展开）”。
- `.fail` 字号更小，只放失败条件，避免喧宾夺主。
- 如果用户要求更像 Claude，优先调低边框对比和阴影，不增加装饰图形。
